FlowMaster

专业的网络流量监控系统

Version 1.0.0 MIT License

✨ 主要特性

实时监控

提供实时流量数据监控，支持自动刷新

多维度统计

支持分钟、小时、日、月、年多个时间维度的数据统计

响应式设计

完美支持移动端，随时随地查看监控数据

🚀 快速开始

curl -fsSL https://raw.githubusercontent.com/vbskycn/FlowMaster/main/install.sh | sudo bash

一键安装命令将自动完成所有配置

🛠️ 技术栈

后端框架

Node.js + Express

前端框架

Vue 3 + Bootstrap 5

监控工具

vnstat

进程管理

PM2

📖 使用说明

1启动服务
flowmaster start
2停止服务
flowmaster stop
3重启服务
flowmaster restart
4查看状态
flowmaster status
5卸载服务
flowmaster uninstall
